# Data Analytics University Project Overview

- Created a specification for a business analytics system for use by senior executives.
- Researched current analytics challenges in the pharmaceutical industry.
- Detailed metrics, KPI and objectives the analytics system would meet.
- Derived KPI formulas for use in dashboards.
- Showed how the analytics system can use descriptive, predictive, diagnostic, and prescriptive analysis.
- Created datasets in Python for use in dashboard creation.
- Implemented Power BI dashboard solution to demonstrate use case.

# Achievements

- Awarded high distinction for the project.
- Gain knowledge on current challenges and trends in data analytics within the pharmaceutical industry.
- Project feedback: "Dashboards are specified at an implementable level with clearly defined KPIs, figures, filters, and explicit drilldown paths. Visual behaviour is illustrated through worked examples that 
show how executive decisions are informed."
- Experience generating datasets in Python and creating dashboards using Power BI.
